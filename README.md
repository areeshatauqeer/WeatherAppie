# Weather App

A simple React Native Weather App built with Expo and WeatherAPI.

## Features

- Search weather by city
- Real-time weather data
- Temperature
- Humidity
- Wind speed
- Weather conditions

## Built With

- React Native
- Expo
- WeatherAPI
